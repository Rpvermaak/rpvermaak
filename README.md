## Projects

**[VeldSentry — Crop Health Monitoring Pipeline](https://github.com/Rpvermaak/sifra-potato-yield-analysis)**
An early warning system for a potato farm in South Africa's Soutpansberg region. Pulls Sentinel-2 satellite imagery via Google Earth Engine, NASA POWER weather data, and soil moisture modelling to produce daily field health scores across 50 fields. Includes a Streamlit dashboard, SMS alerts via Africa's Talking, automated PDF reports, and Docker deployment. Built for and delivered to a real client.

`Python` `Google Earth Engine` `GeoPandas` `Streamlit` `Sentinel-2` `Docker` `Africa's Talking API`

**Overberg Property Analyzer** *(coming soon)*
Scraped 685 real property listings from Property24 across the Overstrand region using Apify, cleaned and combined the data, then trained a Random Forest pricing model. A Streamlit app lets you enter property details and get a predicted market value, valuation assessment, and comparison against similar listings. R2 = 0.45 on real regional data.

`Python` `scikit-learn` `Streamlit` `Plotly` `Apify` `pandas` `Docker`

**Teacher Progress Tracker** *(coming soon)*
A locally-hosted Flask web app built for GrapeSEED English language teachers. Teachers log daily student observations, then the app uses the Claude API to generate professional bi-monthly progress reports — structured into curriculum sections and translated into Korean for parents. Deployed to real teachers across multiple classes.

`Python` `Flask` `Claude API` `HTML/JS` `JSON` `cross-platform packaging`

**[Bike Share Analysis](https://github.com/rpvermaak/bike-share-project)**
Exploratory data analysis on a public bike share dataset, examining ridership patterns, trip duration, and usage differences between casual riders and annual members.

`Python` `pandas` `Jupyter`
